Overview

  • Markdown是一种轻量级标记语言，它以纯文本形式(易读、易写、易更改)编写文档，并最终以HTML格式发布。

Markdown

基础

标题

  • 使用#数量，可表示1-6级标题

目录

  • 使用[TOC]标记来插入目录

分割线

  • 使用** 或者 — 或者 ___ 或者

换行

  • 行末尾加两个空格

引用

  • 使用>表示一级引用，>>表示二级引用，>>>表示三级引用。

    一级引用

    二级引用

    三级引用

列表

  • 使用-、+或*作为标记。

强调

  • 使用**加粗文本** 或者 __加粗文本__。
    • 加粗文本 或者 加粗文本
  • 使用*斜体文本* 或者 _斜体文本_
    • 斜体文本 或者 斜体文本

图片与链接

  • 图片：使用 “![图片文本(可忽略)](图片地址)” 表示
  • 链接：使用“![链接文本](链接地址) ” 表示 
    - 行内链接：[SkySeraph's Blog](http://skyseraph.com)  
- 参考链接：\[SkySeraph's Blog]\[url]，其中url为链接标记，可置于文中任意位置，效果[SkySeraph's Blog][url]。   
- 自动链接：直接使用`<>`括起来<http://skyseraph.com>

脚注(注解)

  • 使用[^]来定义脚注
    • 这是一个脚注的例子^1
