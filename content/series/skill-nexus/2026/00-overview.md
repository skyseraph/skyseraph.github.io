## What is SkillNexus

SkillNexus 是一个围绕 **AI Agent Skills** 的专栏。

在 LLM Agent 时代，"Skill" 是 Agent 调用工具、完成任务的最小执行单元。它不是 prompt，不是 system prompt，而是一段**可执行、可复用、可评测、可进化**的能力描述。

SkillNexus 关注三类核心问题：

1. **Skill 如何构建** — 工具定义、prompt 编排、状态机设计
2. **Skill 如何评测** — 覆盖率、召回率、鲁棒性、跨模型一致性
3. **Skill 如何进化** — 从人工设计到自动合成，从单 agent 到多 agent 协作

---

## Why This Matters

Claude Code、SWE-agent、AutoGPT 这一类 Agent 框架正在重新定义"软件编写"的方式。但它们的底层能力并不是魔法——它们是一组精心设计的 **Skills**（工具调用、任务拆解、代码生成、结果验证）。

理解这些 Skills 的设计哲学，才能真正用好甚至超越这些框架。

专栏将从 **论文解读 + 工具拆解 + 实战笔记** 三个维度，持续追踪 AI Native Tools 与 Agentic Workflows 的前沿进展。
