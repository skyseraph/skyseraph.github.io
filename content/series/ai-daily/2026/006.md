## 📄 论文速递

**1. Memanto: Typed Semantic Memory with Information-Theoretic Retrieval**

Memanto 挑战"知识图谱复杂度是 Agent 记忆必要条件"的固有认知，提出基于信息论的语义记忆层。在 LongMemEval 和 LoCoMo 评测中分别达到 **89.8%** 和 **87.1%** 的准确率——超越所有混合图和向量系统，同时仅需单次检索查询、零摄入延迟、极低运维复杂度。

- **创新点**：13 类预定义记忆分类 + 自动冲突解决 + 时序版本控制，基于 Moorcheh 信息论搜索引擎实现确定性亚毫秒级检索
- **影响**：为生产级 Agent 记忆系统提供高保真、低开销的新架构选择

> [arXiv:2604.22085](https://arxiv.org/abs/2604.22085) · 发布时间：2026-04-28

**2. MolClaw: 药物分子筛选与优化的自主 Agent**

MolClaw 通过三层层级技能架构（工具级→工作流级→学科级，合计 70 项技能）统一 30+ 领域资源，实现药物分子评估与优化。MolBench 基准涵盖 8-50+ 连续工具调用任务，实验证明结构化工作流编排是 AI 驱动药物发现的主要能力瓶颈。

- **创新点**：三层层级技能架构，跨 30+ 专业工具的标准化 Agent 编排
- **影响**：为复杂多步科学工作流提供可扩展的 Agent 架构参考

> [arXiv:2604.21937](https://arxiv.org/abs/2604.21937) · 发布时间：2026-04-28

**3. Rethinking Publication: AI 产出学术论文的认证框架**

论文提出双层认证框架，区分知识质量评估与人类贡献度评级，将 AI 管道产出论文分为 A（AI 可达）、B（需人类定向）、C（超出当前管道能力）三类，为 AI 加速科研产出的出版伦理提供规范性框架。

- **创新点**：分离知识有效性与贡献归属两大认证功能，可落地于现有编辑基础设施
- **影响**：为 AI 科研产出提供透明、公平的出版路径

> [arXiv:2604.22026](https://arxiv.org/abs/2604.22026) · 发布时间：2026-04-28

**4. KARL: 通过知识边界感知 RL 减少 LLM 幻觉**

KARL 提出知识边界感知强化学习方法，在 LLM 中通过 RL 显式建模已知与未知边界，从而减少幻觉问题。实验表明该方法在多个幻觉评测基准上优于现有方法。

> [arXiv:2604.22779](https://arxiv.org/abs/2604.22779) · 发布时间：2026-04-28

**5. Stochastic KV Routing: 自适应深度缓存共享**

提出随机 KV 路由机制，实现自适应深度方向缓存共享，优化 Transformer 推理中的内存使用效率。

> [arXiv:2604.22782](https://arxiv.org/abs/2604.22782) · 发布时间：2026-04-28

**6. Parameter Efficiency Is Not Memory Efficiency: 设备端 LLM 微调重新思考**

挑战传统观点：参数效率不等于内存效率。论文重新审视设备端 LLM 适配的微调策略，提出新的评测视角。

> [arXiv:2604.22783](https://arxiv.org/abs/2604.22783) · 发布时间：2026-04-28
