## 🔥 焦点事件

**1. Anthropic 将 Claude Code 从 Pro 套餐下架**

4 月 22 日，Anthropic 悄然将 Claude Code 从 Pro 订阅页面移除，原本标注"已包含"的条目变为明确的"✗"。The Register 将此解读为 Anthropic 在测试用户反应。此举与 Claude Opus 4.7 同日发生，外界猜测 Claude Code 将转为独立付费产品。

> 来源：[The Register](https://www.theregister.com/2026/04/22/anthropic_removes_claude_code_pro/) · 发布时间：2026-04-22

**2. Claude Opus 4.7 正式发布，强化编程与视觉能力**

Anthropic 发布 Claude Opus 4.7，重点升级方向为高精度代码生成、高分辨率视觉理解和长时 Agentic 工作流。同步上线 Managed Agents 公测版，提供安全沙箱、内置工具集和 SSE 流式输出。可通过 Claude.ai、API 及主流云平台访问。

> 来源：[AI News](https://www.ainews.com/p/anthropic-launches-claude-opus-4-7-with-stronger-coding-vision-and-ai-security) · 发布时间：2026-04 月

**3. OpenAI-Anthropic 企业客户争夺战升温**

Axios 4 月 21 日报道，OpenAI 正积极动员咨询合作伙伴从 Anthropic 手中抢回企业客户，双方均在筹备 IPO。OpenAI 本月已连续发布 Codex 按量计费（4 月 2 日）、Agents SDK 新版（4 月 15 日）、"Codex for everything"（4 月 16 日），全面强化企业侧产品线。

> 来源：[Axios](https://www.axios.com/2026/04/21/openai-anthropic-enterprise-rivalry-heats-up) · 发布时间：2026-04-21

**4. Anthropic 与 Google、Broadcom 扩大算力合作**

Anthropic 宣布与 Google 和 Broadcom 扩大合作，将在 2027 年引入数吉瓦级下一代 TPU 算力。此举被视为为 Claude Mythos（据报道参数量达 10 万亿）的训练和推理做基础设施储备。

> 来源：[Anthropic](https://www.anthropic.com/news/google-broadcom-partnership-compute) · 发布时间：2026-04 月
