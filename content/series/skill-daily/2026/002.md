## 项目详解（今日榜）

### 1. JuliusBrussee/caveman

| 字段 | 内容 |
|------|------|
| 定位 | 穴居人语法 Skill，砍掉 65% Token |
| Stars | 85248 |
| 语言 | JavaScript |
| 更新 | [近期活跃]（日增 1052 星） |

**核心功能**
- 将 Claude Code 交互精简为"穴居人式"极简语法，大幅削减 Token 消耗
- 以更少 Token 完成同等任务，降低 API 成本与上下文压力
- 与主流编码 Agent（Claude Code、Codex、Cursor 等）兼容

**亮点**：极致反共识——不是给 Agent 加能力，而是教它"少说话多干活"，日增千星验证了开发者对 Token 成本的痛点共鸣

**局限**：极简语法可能牺牲细微意图表达，复杂任务场景下可读性与准确性需验证

**链接**：https://github.com/JuliusBrussee/caveman

| Stars | 24279 |

- 将用户意图转化为页面操作（点击、填写、导航等）
