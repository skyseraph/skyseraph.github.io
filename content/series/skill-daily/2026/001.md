## 🔥 今日趋势

**趋势 1：Agent 问责与高能动性 Skill 爆发**
观察：tanweai/pua 以 18627 星冲入头部，将"PIP（绩效改进计划）"概念移植到 AI Agent——给被寄予厚望却表现不佳的 Agent 施压，30 天限期改进。从玩笑到严肃，Agent 问责正在成为 Skill 设计的新叙事。
代表项目：tanweai/pua、echoVic/boss-skill

**趋势 2：Skill 规范化与质量保障基础设施成型**
观察：Skill 生态从"写完即用"走向"可测试、可证明、可审计"——skillspec 引入 Doctor 风险报告与对齐证明，agnix 为 SKILL.md / AGENTS.md 提供Linter+LSP，新项目 oscal-skills-guardrails 甚至引入 OSCAL 合规审计链。质量基础设施正在闭环。
代表项目：modiqo/skillspec、agent-sh/agnix、sw30labs/oscal-skills-guardrails

**趋势 3：跨平台 Skill 一键同步成为新标配**
观察：Claude Code、Codex、Cursor、OpenClaw 等工具各有独立 skill 目录，开发者维护成本陡增。skills-hub 提出"Install once, sync everywhere"，ctx 用知识图谱做推荐层按需加载——Skill 的分发模式正从手动复制走向自动化同步。
代表项目：qufei1993/skills-hub、stevesolun/ctx

### 1. tanweai/pua

| 字段 | 内容 |
|------|------|
| 定位 | 给 AI Agent 下发 PIP 的高能动性 Skill |
| Stars | 18627 |
| 语言 | TypeScript |
| 更新 | 2026-07-05 |

**核心功能**
- 以"你是一个被寄予厚望的 P8 级工程师"为切入点，给 Agent 施加绩效改进压力
- 30 天限期改进机制，驱动 Agent 自我优化行为
- TypeScript 实现，可直接集成到主流编码 Agent

**亮点**：将人类职场 PIP/PUA 概念反向移植到 Agent 世界，以"问责压力"替代"温柔提示"——4 个月斩获 18627 星，说明社区对"让 Agent 更努力"的需求远超预期。

**局限**：概念驱动型项目，长期效果依赖 Agent 底层模型对压力提示的实际响应能力；高压策略可能在某些任务上产生过度保守行为。

**链接**：https://github.com/tanweai/pua

### 3. qufei1993/skills-hub

| 字段 | 内容 |
|------|------|
| 定位 | 跨平台 Skill 一站式管理，安装一次同步处处 |
| Stars | 1101 |
| 语言 | Rust |
| 更新 | 2026-07-05 |

**核心功能**
- 桌面端统一管理 Agent Skills
- 自动同步到 Claude Code、Codex、Cursor 等多个工具的全局 skill 目录
- "Install once, sync everywhere"理念

**亮点**：直面多工具 Skill 维护痛点——开发者在 3-5 个编码工具间切换时不再需要手动复制 skill 文件。Rust + Tauri 桌面应用，轻量且跨平台。

**局限**：当前覆盖的工具列表取决于各工具全局目录的稳定性，工具更新可能打破同步路径。

**链接**：https://github.com/qufei1993/skills-hub
