## 🔥 今日趋势

**趋势 1：Agent Harness 范式确立，工程化成主流**
观察：近期涌现的大量项目不再仅停留在基础 Skill 编写层面，而是向“Harness（框架/套件）”演进。开发者倾向于通过统一规则、记忆机制和调度引擎来编排多个 Agent，并引入评估闭环以确保工程质量。
代表项目：zhayujie/CowAgent、ModelEngine-Group/nexent、revfactory/harness

**趋势 2：Spec 驱动开发（SDD）成新宠**
观察：基于规格说明书的代码生成与验证流程在 Skill 生态中迅速走红。通过将开发意图前置为结构化文档，约束 Agent 行为，再生成工作流的做法，大幅降低了复杂 Agent 任务的“黑盒”失控风险。
代表项目：rpamis/comet、modu-ai/moai-adk

**趋势 3：本土化与领域垂直化 Skill 深入**
观察：Skill 生态开始覆盖高度垂直和本土化场景，从通用编程向特定行业知识（如中医、供应链分析、区域营销）延伸，表明 AI Agent 正在深入长尾实际业务。
代表项目：JuneYaooo/nihaisha-nishi-tcm、yan-labs/serenity-aleabitoreddit
