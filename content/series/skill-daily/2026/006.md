## 🔥 今日趋势

**趋势 1：面向真实工程实战的 Skill 生态爆发**
观察：开发者不再满足于通用的 Agent 提示词，开始分享源自真实 `.claude` 目录下的工程级 Skills，如直接开源 AWS Well-Architected 规范的 Playbook，这标志着 Skill 正向标准化、可复用的 DevOps 资产演进。
代表项目：mattpocock/skills、aws-samples/sample-well-architected-skills-and-steering

**趋势 2：业务垂直领域 Skill 细分深化**
观察：从 Go-to-Market（GTM）销售自动化到 Spec-first 软件开发流程，Skill 正深度切入特定业务工作流。开发者通过组合 MCP 和 Skill 解决实际的管线断层问题。
代表项目：getaero-io/gtm-eng-skills、anton-abyzov/specweave

**趋势 3：CLI 分发与安装渠道逐渐完善**
观察：如何把 Skill 分发给 Agent？社区开始关注 Python CLI 自动化安装 Agent Skills 的能力，打通 Skill 仓库到本地 Agent 运行时的最后一公里。
代表项目：TacoTakumi/AgentSquire
