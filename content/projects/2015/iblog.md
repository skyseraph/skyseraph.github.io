本博客使用 Hugo 静态站点生成器构建，自定义主题，通过 GitHub Actions 自动部署到 GitHub Pages。

## 技术栈

- **构建工具**：Hugo
- **主题**：自定义
- **部署**：GitHub Actions + GitHub Pages
- **评论**：Giscus（基于 GitHub Discussions）
