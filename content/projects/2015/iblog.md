从 2015 年第一篇文章到现在，这里记录了我在 AI/LLM、Android 开发、移动架构等方向上的技术积累与思考。博客本身也是一个持续演进的开源项目。

## 技术栈

- 构建：[Hugo](https://gohugo.io/) 静态站点生成器
- 主题：自研 `skyseraph` theme，支持中英双语、深色模式、代码高亮
- 部署：GitHub Actions 自动构建 → GitHub Pages
- 图床：jsDelivr CDN（`static/images/`）
- 评论：[Giscus](https://giscus.app/)（基于 GitHub Discussions）

## 内容方向

- **AI/LLM**：大模型数据工程、大模型评测工程、Agent工程、Harmless工程...
- **APP**：Android、iOS、NFC...
- **架构设计**：软件架构、性能优化、软件重构...
- **工程效率**：工具链、自动化、开发流程
